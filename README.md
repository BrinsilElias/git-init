## git-init
A tiny and simple cli tool to initialize a git repository and more ... ü™¥

## Usage

You will need to have [Node.js](https://nodejs.org/en/) installed on your machine.

```bash
npx git-init
```

Alternatively, you can install it globally:

```bash
npm i -g git-init
```

And then run:

```bash
git-init
```
